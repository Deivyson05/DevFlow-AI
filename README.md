# DevFlow-AI
Sistema de Apoio Inteligente ao Desenvolvimento de Software


### Preparar ambiente
```
    python3 -m venv .venv
    source .venv/bin/activate
```

### Fechar ambiente
```
    deactivate
```

### Instalar
```
    pip install autogen-agentchat~=0.2
```

### Rodar
```
    autogenstudio ui --port 8081
```
